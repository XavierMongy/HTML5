Basic

<!DOCTYPE>

<html> Définit un document HTML

<title> Titre du document

<body> Définit le "corps du document"

<h1> à <h6> Titres

<p> Définit un paragraphe

<br> Aller à la ligne

<!--...--> Ajouter un commentaire

Images

<img> Définit une image

<map> Définit une carte

<area> Définit une zone dans la carte

<canvas> Génére des formes

<figcaption> Définit une légende

<figure> Sert de conteneur

Audio/Vidéo

<audio> Définit un fichier audio

<source> Défini différentes ressources multi-média

<video> Définit une vidéo ou un film

Listes

<ul> Définit une liste non-ordonée

<ol> Définit une liste ordonée

<li> Définit un élément d'une liste

<dl> Liste de définition

<dt> Définit un élément d'une liste de définition

<dd> Décrit un élément dans une liste de définitions

Tableaux

<table> Définit un tableau

<caption> Définit la légende d'un tableau

<th> Définit le titre d'un tableau

<tr> Définit une ligne d'un tableau

<td> Définit les éléments d'une colonne

Formes

<form> Définit un fichier audio

<input> Défini différentes ressources multi-média

<button> Définit un bouton

<select> Crée un menu déroulant

<option> Définit une option dans un menu

<label>permet de donner le focus à un champ de formulaire

<output> représente le résultat d'un calcul dans un formulaire web
